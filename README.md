# Nascer e Pôr do Sol

This app allows the user to find sunrise and sunset times <br>

## **Motivation**

Project developed so that users can manage their work on the beach, or in the countryside. 
<br>

With it you can have the following information: 
- Sunrise 
- Solar noon 
- Sunset 
- Duration of the day
<br>
